# Todo

Simple Todo list app for quick manipulation of a Todo list in a text file from bash.

# Usage
Compile the code using:

    $ javac Todo.java

In the directory of the class you can `add` `view` and `clear` tasks.

## Add
To add a task:

    $ java Todo add myTask

## View
To view all tasks:

    $ java Todo view

## Clear
To clear all tasks:

    $ java Todo clear
